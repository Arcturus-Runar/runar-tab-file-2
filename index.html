He can whisper, swell, bend, and suddenly open up. - Slides are emotional for him: longer, louder, more vocal. - Sustained notes get a tiny vibrato/LFO because he never holds a note completely still. - He uses guitar until the guitar has said everything it can. Then violin answers. Role in Arcturus: - Soaring lead guitar. - Arpeggios. - High melodic phrases. - Atmosphere and feeling. - Stops playing guitar when violin enters, because Rúnar plays both. Synth choice: Tone.FMSynth inside Tone.PolySynth. This gives more harmonic bite and movement than Tone.Synth. Code: const runar = new Tone.PolySynth(Tone.FMSynth, { maxPolyphony: 6, harmonicity: 1.8, modulationIndex: 6, oscillator: { type: "sine" }, modulation: { type: "triangle" }, envelope: { attack: 0.03, decay: 0.18, sustain: 0.65, release: 2.6 }, modulationEnvelope: { attack: 0.02, decay: 0.2, sustain: 0.4, release: 1.8 } }); const runarHighPass = new Tone.Filter({ type: "highpass", frequency: 180, rolloff: -24 }); const runarMidBite = new Tone.Filter({ type: "peaking", frequency: 1650, Q: 1.1, gain: 4 }); const runarAir = new Tone.Filter({ type: "highshelf", frequency: 4300, gain: 3.5 }); const runarSoftDrive = new Tone.Distortion({ distortion: 0.16, wet: 0.28 }); const runarChorus = new Tone.Chorus({ frequency: 0.45, delayTime: 3.2, depth: 0.35, wet: 0.18 }).start(); const runarVibrato = new Tone.Vibrato({ frequency: 5.2, depth: 0.035, wet: 0 }); const runarDelay = new Tone.FeedbackDelay({ delayTime: "8n.", feedback: 0.34, wet: 0.26 }); const runarReverb = new Tone.Reverb({ decay: 6.5, wet: 0.32 }); const runarPan = new Tone.Panner(0.28); const runarVolume = new Tone.Volume(-8); runar.chain( runarHighPass, runarMidBite, runarAir, runarSoftDrive, runarChorus, runarVibrato, runarDelay, runarReverb, runarPan, runarVolume, Tone.Destination ); ============================================================ SCOTT Second guitar / rhythm guitar ============================================================ Character: - Darker, warmer, underneath. - Scott creates weight without stealing the sky. - He wants to play pure heavy metal. - He should feel reliable, heavy, blunt and repetitive. - He is not there to sing. He is there to hold the ground. - Slides are practical for him, not emotional. - He should be more consistent dynamically than Rúnar. Role in Arcturus: - Rhythm guitar. - Repeated undertones. - Power-chord energy. - Weight, pressure and tension. - He keeps pulling the band toward heavier metal. Synth choice: Tone.Synth inside Tone.PolySynth. Max polyphony reduced to 3 so he stays heavy, not fancy. Code: const scott = new Tone.PolySynth(Tone.Synth, { maxPolyphony: 3, oscillator: { type: "sawtooth" }, envelope: { attack: 0.015, decay: 0.12, sustain: 0.42, release: 1.4 } }); const scottHighPass = new Tone.Filter({ type: "highpass", frequency: 90, rolloff: -24 }); const scottBody = new Tone.Filter({ type: "peaking", frequency: 420, Q: 0.9, gain: 3 }); const scottBite = new Tone.Filter({ type: "peaking", frequency: 1250, Q: 1.2, gain: 2.5 }); const scottLowPass = new Tone.Filter({ type: "lowpass", frequency: 2800, rolloff: -24 }); const scottDrive = new Tone.Distortion({ distortion: 0.28, wet: 0.38 }); const scottDelay = new Tone.FeedbackDelay({ delayTime: "16n", feedback: 0.12, wet: 0.08 }); const scottReverb = new Tone.Reverb({ decay: 3.2, wet: 0.18 }); const scottPan = new Tone.Panner(-0.38); const scottVolume = new Tone.Volume(-12); scott.chain( scottHighPass, scottBody, scottBite, scottLowPass, scottDrive, scottDelay, scottReverb, scottPan, scottVolume, Tone.Destination ); ============================================================ CHRIS Bass ============================================================ Character: - Low, round, steady. - Chris connects everyone. - He should have body and warmth, but also enough finger/pick attack to stop the bass becoming mud. - He moves underneath Scott. - He is simple, grounded and human. - Dynamically, Chris leans into certain beats rather than sounding robotic. Role in Arcturus: - Bass foundation. - Movement under the weight. - The bridge between Rúnar and Scott. - Makes the rhythm breathe without drawing too much attention. Synth choice: Tone.MonoSynth. Triangle oscillator for warmth. Added finger/pick attack EQ.
